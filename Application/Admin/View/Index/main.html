  • Think-BASE框架基于上海顶想公司开发维护的MVC开源框架ThinkPHP,遵循Apache2开源协议发布
  • Think-BASE的前端页面基于bootstrap前端框架，Bootstrap是Twitter推出的一个用于前端开发的开源工具包,是一个CSS/HTML框架
  • Think-BASE后台模板的构建使用layui,Layui 是一款采用自身模块规范编写的情怀型前端UI框架,遵循原生HTML/CSS/JS的书写与组织形式
  • Think-BASE在吸取前辈们的开发经验基础上，经过架构优化，取长补短，使得基础架构更加合理，可扩展。
  • Think-BASE借鉴onethink和thinkcmf等优秀框架的基础，对前后台用户模型做了进一步完善。使网站会员和后台管理员分离
  • Think-BASE对一般网站的权限管理和网站前后台会员做了基础的构建，方便广大开发者使用
  • Think-BASE的前端更加通俗易懂，完全解决了一般后端开发人员对前端的烦躁和不熟悉
